# Joe Hunter — Portfolio & Resume

Personal portfolio and interactive resume for **Joe Hunter**.

## Features

- **Dark mode**
- **Expandable job descriptions**
- **Download PDF**
- **Scroll animations**
- **Responsive**
- **Social sharing**

## Tech stack

- Vanilla HTML / CSS / JS
- [Font Awesome 6.4.0](https://cdnjs.cloudflare.com/ajax/libs/font-awesome/6.4.0/css/all.min.css) (CDN)
- Hosted on **GitHub Pages**

## Development

No build tooling required. Edit `index.html` directly and open it in a browser to preview. Push to `main` to deploy.

## Deployment

GitHub Pages auto-deploys from the `main` branch. The site is live at [jrhunter3.github.io](https://jrhunter3.github.io/).
